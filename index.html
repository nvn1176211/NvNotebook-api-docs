Welcome to Notebook api documentation!

POST Create a new user

http://127.0.0.1:8000/api/users
Query parameters
Name Required Value Description
name Yes String User's name
email Yes String User's email
password Yes String User's password

Example response

Status: 200

{
    "name": "user2",
    "email": "user2@gmail.com",
    "updated_at": "2024-10-08T15:00:34.000000Z",
    "created_at": "2024-10-08T15:00:34.000000Z",
    "id": 5
}
